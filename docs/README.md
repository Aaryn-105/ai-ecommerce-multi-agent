# Docs

Long-form reference material that does not belong inside the application source tree.

## `figma-reference/`

Screenshots of the Figma designs used during the frontend rebuild. These are
**inputs**, not generated artifacts — keep them under version control so any
contributor can see the source design intent.

## `reference-reports/`

PDF snapshots cited in design conversations as the gold standard for report
quality (formatting, charts, polish). Use these as a benchmark when judging
whether a regeneration looks correct.
